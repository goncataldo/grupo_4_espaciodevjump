html

Aprendemos a estructurar

HTML es el bloque básico con el que está construido internet. Todas las páginas web utilizan HTML. son lenguajes que incorporan al texto marcas o etiquetas que luego son interpretadas para darle información extra sobre la estructura del texto. En el caso de HTML, este será interpretado por los browsers, que tambien presentaran el código en forma gráfica. Es el primer paso en la carrera de DevJumper.
css

Aprendemos a dar estilo

CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web, e interfaces de usuario escritas en HTML. Este va de la mano de HTML y es el segundo paso en la carrera de DevJump
js

Aprendemos a dar funcionalidad

JavaScript es un lenguaje de programación interpretado. e define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico. Se utiliza principalmente del lado del cliente, implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas. Es el tercer paso en la carrera DevJump y uno de los mas importantes.
