Introducción

¿Qué es CSS y para qué sirve?
CSS significa Cascade Style Sheet o hoja de estilo en cascada y nos permite crear reglas que definen propiedades visuales para nuestros elementos.

o En el código de ejemplo definimos que todos los elementos span de nuestro documento/s deben tener la tipografía de color blanco.

Ejemplo:

span {
color: white;
}

o De esta definición vemos que tenemos una forma de seleccionar elementos y que podemos aplicar una propiedad para modificar la forma en que se ve el contenido de la etiqueta seleccionada.
o En CSS todo el tiempo vamos a estar utilizando estos dos conceptos:
o Seleccionar elementos
o Definir como queremos que se vean
o Según como utilicemos CSS podemos definir como se ve un elemento, un documento o todo nuestro sitio
